This is my collection of images and rulebooks for several TCGs and CCGs. Each game is packed in .rar files for each set and one for promos. Even like this there is ~ 1000 rar files. Some of thsse come from this site while others have been found other places, and I would like to thank the original uploaders. pulling out these images would be a daunting task so they are still included.

There are rulebooks for 61 games.

The quantity and quality of images varies, spongebob has 2 high quality images while G I Joe is a complete low- medium quality image set.

Card naming has no consistency, some are labeled by number others by card name, etc.
