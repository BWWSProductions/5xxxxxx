Release Description

ZIP Password Recovery Magic is an easy-to-use program that can help you
to recover lost passwords for zip archives.

ZIP Password Recovery Magic provides brute-force and dictionary recovery
methods, you can pause and resume recovery job easily

ZIP Password Recovery Magic has an easy to use interface. All you need
to recover your password is just to add your file to the operation
window

Installation

Just unpack&install. Register with the serial in serial.txt
