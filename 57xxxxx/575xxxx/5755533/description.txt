Installazione
Masterizzate o montate il file immagine con le daemon tools lite. Installate il gioco e buon divertimento.

Per avere menu e sottotitoli in italiano aprite il driver virtuale su cui avete montato il file immegine, andata nella cartella conv.ITA ed eseguite il file [CNV]Shutter-Island[TESTO][TBH] facendo attenzione ad indicare alla conversione il giusto percorso del gioco (che dovrebbe essere C:\Programmi\ShutterIsland-1.0.0 ).
