Title: Sanity: Aiken's Artifact
Developer: Monolith Productions
Publisher: Fox Interactive
Genre: Action
Released: 2000
Description:
Sanity is an action game with adventure and puzzle-solving elements. There is also an RPG element present in card-collecting and upgrading the talents. The game is heavy on combat, which is mostly ranged, with Cain either shooting or using the psychic powers. The talents are classified into groups, such as Fire, Sun, Illusion, Science, Demonology, etc. In each group, there are many talents Cain can learn, which range from offensive and supporting combat spells to abilities needed to solve puzzles (such as levitating, pushing heavy objects, etc.)
