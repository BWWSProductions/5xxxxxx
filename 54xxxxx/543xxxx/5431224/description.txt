                           Description                            
                                                       
                                                                        
                                                                          
                                 GekoTask                                 
                      All your tasks on a fingertip!                      
                                                                          
     Do not miss any task! Simply create your to-do list and have your    
                  dates automatically set in your agenda.                 
                                                                          
      GekoTask is the application for smartphones that allows you to      
       manage all your tasks: easily and with only one finger. Add,       
     modify or delete tasks with few clicks and synchronize them with     
                 Microsoft Outlook (on smartphone and PC).                
                                                                          
          Download and test the trial version for 15 days without         
     limitations, and have fun with the picker and with the advantages    
                               of GekoTask.                               
                                                                          
                                 Features                                 
                                                                          
                              Sort and Filter                             
                                                                          
                            GekoTask is smart!                            
                                                                          
      GekoTask allows to sort tasks by status, due date, start date,      
     subject, priority and category. Moreover, user can select one or     
      more categories to filter tasks list in order to show only the      
                             preferred tasks.                             
                                                                          
                                                                          
                           Smart user interface                           
                                                                          
                       GekoTask is finger-friendly!                       
                                                                          
     GekoTask is designed to achieve the best level of ergonomics and     
      ease of use. All the functions are accessible without using the     
          stylus; the interface is simple and intuitive; the time         
      scheduling is done by using the kinetic picker. Complete a task     
            simple clicking on check button in the tasks list.            
                                                                          
                                                                          
                         Personalization of Tasks                         
                                                                          
                        GekoTask is full optional!                        
                                                                          
     GekoTask allows you to set a category to your task. You can also     
        customize your list of categories and assign the following        
         attributes: priority, status, sensitivity, recurrency and        
                                 reminder.                                
                                                                          
                                                                          
                            Invitation over SMS                           
                                                                          
                       GekoTask is social oriented!                       
                                                                          
      With GekoTask you can forward your invitation via SMS. You can      
     invite your friends or colleagues simply by selecting their name     
       from the contact list in your smartphone; they will accept or      
        decline and the date will be automatically created in their       
      agenda. You will receive a reply notification. (This option is      
      possible only if GekoTask is installed also in the addressee's      
                                smarphone).                               
                                                                          
                                                                          
                 Fitting for different screen resolutions                 
                                                                          
                        GekoTask is self adaptable!                       
                                                                          
     GekoTask supports the following screen resolutions: QVGA, WQVGA,     
      VGA, WVGA in "portrait" mode. The program automatically adapts      
                  the graphic to your screen resolution.                  
                                                                          
                                                                          
                             Open to new skins                            
                                                                          
                         GekoTask is as you like!                         
                                                                          
      GekoTask supports your graphical skins. You can personalize the     
                     look of the program as you want.                     
                                                                          
                                                                          
                      Support of unlimited languages                      
                                                                          
                          GekoTask is worldwide!                          
                                                                          
       GekoTask is delivered with the following languages: English,       
       Spanish, French, German, Italian, Russian, Dutch, Norwegian,       
        Swedish, Greek, Czech, Danish, Polish, Ukrainian, Romanian,       
      Chinese (traditional and simplified), Japanese. You can add any     
        other language simply by uploading a resource file with the       
                               translation.                               
                                                                          
                                                                          
                         Windows Mobile compliant                         
                                                                          
                      GekoTask is what you expected!                      
                                                                          
     GekoTask supports all touchscreen devices based on Windows Mobile    
                                5.x and 6.x
