The Windows registry is a database which stores settings and options for the operating system for Microsoft Windows 32-bit versions and 64-bit versions. It contains information and settings for all the hardware, software, users, and preferences of the PC. Whenever a user makes changes to "Control Panel" settings, or file associations, system policies, or installed software, the changes are Reflected and stored in the registry.
