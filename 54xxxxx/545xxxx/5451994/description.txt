Gorky-18: Male's Work----------------------------------------------------------------

Old famous RPG/Isometric Gorky-17, creatively revised by Dmitry "Goblin" Puchkov.

This ISO image was made from the original disk, some extras (4 demo games, DirectX 7 and Windows Media Player 6.4) were ripped, no game files haven't been touched.

Mount image (it will need at each launch), install the game and enjoy!

                      !!!Attention, Russian language only!!!

Goblin's opening speech (in Russian): http://www.snowball.ru/gorky17/?page=gorky18



Gorkij-18: Muzhskaya Rabota----------------------------------------------------------

Staraya dobraya RPG Gorkij-17, tvorcheski pererabotannaya Dmitriem Puchkovym "Goblinom".

ISO obraz sdelan s original'nogo diska, dopolnitel'nye materialy (4 demoversii drugikh igr, DirectX 7 i Windows Media Player 6.4) vyrezany, nikakie igrovye fajly ne byli zatronuty.

Smontirujte obraz (on budet trebovat'sya pri kazhdom zapuske), ustanovite igru i naslazhdajtes'!

Vnimanie, igra tol'ko na russkom yazyke :-)

Vstupitel'naya rech' Goblina: http://www.snowball.ru/gorky17/?page=gorky18
