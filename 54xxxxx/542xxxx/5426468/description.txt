How to Install

- Install Starcraft 2 Beta. (let it update versions with blizzard)
- Install the Crack to the folder that SC2 was installed to (Program Files/ Starcraft II Beta), it will prompt you to replace files - say yes to all.
- Extract the maps, it can be anywhere. I use desktop.
- If you want to change the difficulty, copy the difficulty file to StarCraft II BetaModsLiberty.SC2Mod
- Run Lazylaunch, it will prompt you to choose a map, select whichever you desire.
- Enjoy Starcraft 2 Beta !



Note ----
Some systems may reach high temperatures and overheating conditions while running StarCraft II. This is mainly due to the video card rendering the screens at full speed. As a workaround, there are two lines that you can add to your DocumentsStarCraft II Betavari ables .txt file to limit this behavior.

Frameratecapglue=30
Frameratecap=60

The frameratecapglue controls the framerate at the menu screens. The frameratecap controls the framerate on all other screens. You may adjust these numbers to your liking.
