Call of Duty World at War PATCHES v1.0 - v1.7
-----------------------------------------------------------

This is not the game, only the patches.
You'll need to have the actual game installed.

These patches are included:
v1.0 to v1.2
v1.2 to v1.4
v1.4 to v1.5
v1.5 to v1.6
v1.6 to v1.7

HOW TO INSTALL:

1. Unpack the archive with winrar. (Just google it if you don't have it)

2. Just run the first patch (CoDWaW-1.2-PatchSetup). When completed, install the next patch (CoDWaW-1.2-1.4-PatchSetup) and so on...
