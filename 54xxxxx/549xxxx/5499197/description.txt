Install Notes
-------------------------------------------------------------------------------------

1. Install as trial version without any serial number
2. When installation completes, restart computer, disconnect from internet and launch Ecotect. It will ask you to 'Run' or 'Activate' - select 'Activate'
3. Enter 999-99999999 as serial number
4. Launch X-Force keygen
5. Press 'Mem Patch' (it will give you confirmation of memory patching)
6. Copy 'Request Code' from license screen and paste it into keygen, press 'Generate'
7. Copy 'Activation Code' from keygen and paste it to license screen
8. Click 'Next', and it's DONE!
