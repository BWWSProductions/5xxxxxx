Xbox 360 Wave 4 Boot Disc
-------------------------

If when you try to play the latest games you get the 'Play DVD' 
message instead of 'Play Game' you need this DVD

Instuctions
-----------
1. Burn this ISO to a Dual Layer DVD (all data on 1st layar)
2. Insert Disc into 360
3. When it says 'play DVD' ejext the disc
4. Insert your wave 4 game disc
5. You should now see 'Play Game'.... So Play


NOTE: You will need to reload this disc before booting each game
      
WARNING: DO NOT play online. Stealth will be disabled
