Sid Meier's Alpha Centauri and Alien Crossfire:
Contains the original Sid Meier's Alpha Centauri,
plus the expansion Alien Crossfire.

This release does not need to be burned to CD.
It is complete with all music/voice/movies and works flawless from the hard-drive.

Manuals from the games are included as pdf files in the game folder.
Have fun with this classic!

Install notes:
Extract the rar archive to a folder of your choice.
You launch the game from axstart.exe.
You can also run Sid Meier's Alpha Centauri from terran.exe or Alien Crossfire from terranx.exe.
