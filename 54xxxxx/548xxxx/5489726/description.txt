A collection of useful tools for every day use in Windows environment.

This colecction is centered on TCM (Total Commander Starter), which prepares an environment for starting Total Commander with all the tools, taking care of the program settings (each user has his own settings).

TCM.exe copies itself to windows directory, so you can launch TCM from the command line.
