Windows XP Professional x86 SP3 HU Revived 20100406

Tartalom:

- magyar Windows XP
- frissitesek 2010.04.06-ig
- nem kell aktivalni
- nem kell telepitesi kulcs
- nem kell a WGA-t letiltani

A Microsoft Update aktivalasahoz a Start Menubol el kell inditani a Windows
Update-et, majd amikor megjelenik a Microsoft Update logo, az "Inditas"-sal
lehet engedelyezni. Az elejen sokaig tarthat a frissitesek keresese, csak
turelmesen.

Ha a frissitesek kovetkezteben ellenorizni akarja a Windows eredetiseget, nem
kell megijedni, le lehet futtatni, utana megkoszoni, hogy eredeti Windows-t
hasznalsz.

Ha az alapveto beallitasok keszen vannak, celszeru egy korlatozott felhasznaloi
fiokot letrehozni a mindennapi hasznalatra, ugyanis alapbeallitaskent
rendszergazdakent leptet be az XP.
