Fallout 2 Revived Data Files

These 2 files are required for the Fallout 2 Revived Installer. These files are
common and won't change after any update, so you need to download these only
once, and updates to the Revived Installer will require much less bandwith
and disk space.

Also download the Revived Installer. After you ran it, copy these 2 files to
the install folder, otherwise you won't be able to run the game.
