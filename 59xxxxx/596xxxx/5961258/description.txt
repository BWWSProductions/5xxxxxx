Description: OpenSuse is popular Linux based operating system. It is open source, and sponsored by Novell. It provides most things that people expect in a modern operating system.

File: This is the livecd image of OpenSuse 11.3. It can be run live, to test it out, without any changes to one's hard drive. Then you can install it to the hard drive.

Instructions: Burn this .iso image to a disk with a iso recorder(e.g. isorecorder). Then boot from the disk, and installation is self explanatory.
