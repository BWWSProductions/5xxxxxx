Linux Mint 10 "Julia" - DVD (64-bit)

This is Linux Mint 10, codename Julia, based and compatible with Ubuntu 10.10 Maverick Merkaat and its repositories.

New features

Based on Ubuntu 10.10 Maverick Merkaat, Linux 2.6.35, Gnome 2.32, Xorg 7.5.

Linux Mint 10 "Julia" features a lot of improvements and the latest software from the Open Source World.

Featured improvements in this release: 30,000 applications catalogued and reviewable both online and in the new software manager, brand new incremental backup tool for both data and software selection, menu transparency and editable items, look & feel improvements.
