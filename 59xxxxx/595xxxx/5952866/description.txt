Description: Updates the game 2.0.1.4 - 2.0.5.0

Installation:
To update the game to version 2.0.1.4 or higher to version 2.0.5.0 you need to download all the files and put them in one folder, then run the file l4d2_global_patch_2050.exe, and then follow the instructions.

To update the game to version 2.0.4.5 or higher to version 2.0.5.0 you need to download the file only l4d2_global_patch_2050.exe and follow the instructions.
