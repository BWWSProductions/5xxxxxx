Steps:

    1. Run VBT-Install

    2. Once finished installing, Run VBT-Update 1

    3. Once finished installing, Run VBT-Update 2

    4. Once finished installing, Run Patch and click Patch

    5. In the new windows explorer window navigate to the install directory:
                            Example:  C:\Program Files\Beat Kangz\Virtual Beat Thang

    6. Click on VirtualBeatThang.exe

    7. You are now Patched... Enjoy
