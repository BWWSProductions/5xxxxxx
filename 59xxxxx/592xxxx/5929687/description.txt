
ThePirateApp is an open-source project which allows you to download media through russian servicedirectly. This piece of software comes malware-free and everyone is invited to contribute towards future releases and the improvement of this software itself.

IMPORTANT: This project including the binaries and sources is only for educational and information purpose and proof-of-concept. This should not be used to any illegal actions. If you cannot accept the following, please close the application now.

Pirate is a graphical user interface for downloading files from vkontakte.ru developed in VB.NET. The application provides the user with an easy-to-use interface using the open source (with a bit of customization) XPTable component. The binary and source is available from Google Code under the name "pirateapp" at the URL: http://code.google.com/p/pirateapp/
