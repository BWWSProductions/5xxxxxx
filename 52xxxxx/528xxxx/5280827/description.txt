This is AAA (all aircraft arcade) mod for IL-2 1946 sturmovik game. Please note, that this mod's version is 1.1


Instalation:
1. Unrar with WinRAR or 7zip
2. Locate your IL-2 directory
3. Move all files (but not replace!!!!) to the il2 directory
4. Enjoy!




P.S. If you want to mod working, you need first you need to install these mods:

Mod activator or ENG_SOUND_MOD
Wrapper.exe
