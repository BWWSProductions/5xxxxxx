Product Description:
VirtualHostX 2.0 is the easiest way to host multiple 
websites on your Mac. It's the perfect solution for web 
designers working on more than one project at a time. 
(Aren't we all?) No more nesting folders or asking the 
programmer across the cubicle for help. With 
VirtualHostX you can easily create and manage Apache 
virtual hosts with just a few clicks.

Features
--> Automatically configures your Mac's web server 
settings
--> Works with MAMP, XAMPP, and Apple's built-in web 
server
--> Share your hosts with other users on the same local 
network
--> Supports custom Apache directives
--> Backup and Restore your web server settings
--> Create a complete web development environment on 
your Mac

What's New?
--> Added preference to hide pop-up help messages.

Installation:
--> Open folder & mount app
--> Drag app to your applications folder
--> Use serial from .txt file to register program
--> Done!
