Product Description:
GUI for FFmpeg (FFmpeg - a set of libraries open source software that can record, convert and transmit digital audio and video in various formats.)

Easy to use graphical interface
Using small system resources
Multitasking processing
Options
Extract audio from video
Converting video: MP1, MPEG4, H.264, XviD, AVI, WMV, MOV, FLV, and others.
Convert audio: MP2, MP3, M4A, AAC, FLAC, alac, WAV, WMA, AC3, OGG and others.
Converting to IPod, iPhone, PSP, PS3, XBOX360, ...
Uses FFMPEG for en / decoding. High quality guaranteed
