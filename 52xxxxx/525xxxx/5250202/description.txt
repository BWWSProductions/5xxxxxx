Dingoo a 320 Maetel 1.2 firmware.
www.maetel.snk-neofighters.com

------------------------------------------
Maetel 1.2 is a modification of Dingoo Digital China 1.20 firmware.

Changes :
------------------------------------------

*3D games folder renamed as "Dingoo games"
*Interesting games renamed as "Roms"
*U disk renamed to "Internal Memory"
*Supports custom themes
*PSP icons and Win 7 backgrounds added
*English language as default
*Original Bootscreen/Logoff
*GBA Loading Screen changed
------------------------------------------

Games and apps added :
------------------------------------------

*430Mhz Overclock patch
*7 Days Salvation English
*Ultimate Drift English
*Decollation Warrior English
*Yi-chi King fighter English
------------------------------------------

Emulators Added and updated (12/31/09):
------------------------------------------

*ProoSystem = Atari 7800
*LynGOO = Atari Linx (Needs lynxboot.img in "game" folder, google it !)
*WS = Wonderswan
*SmsGGOO = Sega Master
*O2EM A320 = Odyssey2 / Videopac+
*NGP = NeoGeo Pocket
*MVS = NeoGeo Emulator with "Test Swith Enabled"
*GnuBoy = GameBoy/Color
*DOOEngine = PC Engine
*ColecOO = Colecovision
*MD = Mega drive with "Double buffer technique"
