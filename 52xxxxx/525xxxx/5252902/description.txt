Here is kzhack v3 public version with a 70 strafes script. Only works on steam and airaccelerate 100.

Instructions:

1. RUN CS (wait until its loaded) 
2. OPEN KZH (it will close imediatly that's good) 
3. It will appear a message on console 
4. JOIN SERVER 
5. WRITE kzh_exec longjump.cfg ON CONSOLE 
6. write bind key +sw23.str (replace key with the key you want) 
7. then press the key you binded and it will do strafes 
8. DONE!
