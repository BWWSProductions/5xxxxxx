The Launcher allows you to find and see a screenshot and read about each game before you select to play it, it will go back to the Games Launcher when you exit the game.

Mega Games Pack 1.23 and MGP 4 games will get detected and be available from the v7 Games Launcher.

(Note: If having trouble installing previous collections, remove the "C:WindowsLastXPs.ini" then it'll install those games fine. This was a side effect with the previous installer.)

I have included all source code and Tools I used to make this pack, so people can make their own collections without having to learn too much.

Start Menu sorting is available to LastXP users else it defaults to one folder per game installed. You can select a startmenu method using the LastXP Settings from the Autorun menu or Via the LastXP ssWPI menu.
