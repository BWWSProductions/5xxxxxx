This is a ripped verison of starcraft broodwar.

How to play

1. Unzip the starcraft Original+Broodwar rip.zip.

2. Open the install.exe A dialogue saying that the installation has completed
   successfully will appear when Finished.
 
3. Open the install1.exe If successfull, a command prompt will appear for a
   second before it closed.

4. Open the install2.reg and press ok when a dialogue appears.

5. Open the broodwar.exe to play.


* This only works from window 95 to window xp.
