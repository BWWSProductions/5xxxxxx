Due to the small size, xPUD comes with limited drivers and applications. You can extend the functionalities of xPUD by using Opt-Get software plugin system.

=== Instructions ===

Download these Opt plugins, and put them under opt/ directory of your bootable device, they'll be auto loaded while xPUD is booting.

=== About xPUD ===

xPUD is an unique Linux distribution with a simple user interface on top, including a web browser and a media player.

xPUD is fast booting : average booting within 10 seconds. It's light : core image is weighing less than 35MB. IT's easy to use with a very simple user-interface.

More information on http://xpud.org
