1.1 Installing / Uninstalling EPC version 15.00 
============================================== 

To install EPC version 15.00 : 
----------------------------- 
Run SETUP.EXE from the root of your CD-rom drive. 
(see CD-cover for step-by-step instructions) 

To uninstall EPC : 
------------------ 
Select 'Add/Remove Programs' in the Windows 
control panel. Select 'Electronic Parts Catalogue' 
and click on 'Add/Remove..'.

1.3 Important Notes 
=================== 

1. Due to VIN changes (concerning 0 and W Year 
Indication), sometimes you will not be able to 
select parts when using Model or Part. You will then 
receive the following message : 

"It is impossible to select parts via Model or Part 
for the selected car ! Use VIN to select the 
correct parts !" 

Example : 

Accord 4 doors 1996 1.8I 5MT KE 

VIN VIN Ranges 
--- ---------- 
SHHCE7540 0 U 0U000001 - 0U026609 
SHHCE7540 W U WU026610 - WU099999 

Since these are two different Modeltypes, EPC is 
unable to know the correct Modeltype without 
knowing the VIN.
