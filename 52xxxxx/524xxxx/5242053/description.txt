Agama Web Buttons enables you to create professional quality, unique web buttons in a matter of minutes. With Agama web Buttons you can create a variety of 2D and 3D glassy, metallic, plastic and even XP or Vista style Animated web buttons from predefined templates. Base installation of Agama Web Buttons contains 70 packages with more then 700 web button templates for instant use.

Creating your own buttons with Agama Web Buttons is very easy. Only three steps and your Web pages will be enriched with nice, professional web buttons that respond to a mouse over event.

Features:
1. Select button template, which you like.
2. Type the text of your graphics, for example "Home", "Products", "About Us", etc. Optionally customize your graphics by changing the font, text style, color, size, lighting effects. Optionally you can add an icon. To create multi-state "rollover" graphics you can customize both states of web button.
3. Export your graphics to one of these formats: GIF, PNG, JPG, BMP
4. Agama Web Buttons can create HTML and javascript code handling Mouse Over effect. You can choose HTML template for normal web menu (vertical layout) or navigation bar (horizontal layout).
