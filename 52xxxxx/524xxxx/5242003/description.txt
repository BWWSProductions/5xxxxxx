SAPGUI for the Java Environment is a unified SAP frontend for multiple platforms

Mac OS X

Hardware

Intel Core 2 Duo for Mac OS X 10.5 (64-bit, Intel-based Macs)
Intel Core Duo for Mac OS X 10.6 (Intel-based Macs)
RAM: 1GB required, 2GB recommended for better performance
Software

Mac OS X 10.5.7 and newer 10.5.x versions require Java for Mac OS X 10.5 Update 4 incuding Java SE 6 for 64-bit, Intel-based Macs (java.version is "1.6.0_13" as shown in "About SAPGUI") or newer.
Mac OS X 10.6 and newer 10.6.x versions (includes Java SE 6 for 32-bit and 64-bit, Intel-based Macs)
For a table of Apples Java release names and their corresponding Java version strings, refer to Technical Note TN2110.
