*****How to Install*****
-Extract .rar's  (Please ignore the .rar name "music" but still extract as it contains everything)

-First Install OZtopo by the Setup.exe file in the 'Garmin Australia-Topo-OZTopo_v2_1_Roads' Folder

-Then Mount and Install MapSource trip and waypoint manager (in the 'Garmin.Trip.and.Waypoint.Manager.MULTILANGUAGE.ISO-AUXiSO folder) then install later version of mapsouce in the mapsource folder.(If mapsource shows error about oztopo, then install oztop_MS613_patch.exe)

-----------------------------------------------------------------------------------

*****KeyGen******

Enter random UNIT ID number.

Click generate

Select Map Product- scroll down too custom mapset

Enter 5027 in the MapID code feild.

Click The second Generate button.

-----------------------------------------------------------------------------------

*****MapSource*****

Open MapSource program.

Goto>Utilities>Manage map products

Click the 'unlock regions' tab

Click 'Add'

Enter the unlock code which is located on the bottom feild of the Garmin unlock Generator.

Click 'Ok'

Maps should be unlocked :)

-----------------------------------------------------------------------------------

*****Optionals******
-Optionals include nRoute (if you have a gps reciever or gps connected to your pc then you can have live position)

-GPSgate is used to connect to your gps via serial or usb and convert its data to mapping software, eg, google earth, Mapsource
