   Exif wMarker is a simple and user friendly tool to add a watermark to your   
   images. What is a watermark ? Watermark is a text written on your images,   
   this text can be a fixed text written by you or can be the content of exif   
   or iptc metadata fields. Exif wMarker allows you to batch process all   
   images inside a folder and automatically write over them any information   
   taken from exif or iptc metadata fields.                              
                                                                         
   The text appearance can be easily customized, you can change font, color 
   and opacity or set special effects like outline, shadow, glow or emboss. 
                                                                         
   File types supported are Jpeg and Tiff. Iptc tags are read from Jpeg files 
   only.
