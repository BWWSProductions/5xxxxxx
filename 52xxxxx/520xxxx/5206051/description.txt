A Video Tutorial on how to install cwRsync on a server and client and transfer files

cwRsync is a packaging of Rsync and Cygwin. You can use cwRsync for fast remote file backup and synchronization. This archive consists of a 20 minute video tutorial on how to install, configure and transfer data using this product as well as a 2 minute video addendum reflecting comments made by people who saw the first footage.

Server and Client source code used in the demo available from http://www.AlanBonnici.com.
