Torrent Search 3.0 is a torrent search toolbar for Firefox, Internet Explorer and Mac Safari, which enables users to search over 180 top torrent sites and trackers from their Firefox browser the same way they used to search usingform. It allows users to search a single site, search all sites on one page, or open the sites in Firefox tabs.A lot of accessible tools have been added to the Toolbar to make it easy to find your torrent like 'keyword highlighter', 'select toggle' to toggle between search engines by category, and 'toggle by first letter' to easily find the search engine you need.
