Windows XP Professional(X86) with SP3, self activated and integrated with SATA Drivers for the following Chipset Manufacturers :

1. Intel

2. AMD

3. NVIDIA


Instructions : (No CD Key required)

1. Burn the ISO file on to a disc using Nero or PowerISO.

2. Boot the PC/Laptop using the disc.

3. Follow the on-screen instructions for installing Windows XP.


For some Laptops, you might have to change the compatibility mode (AHCI/SATA), once you do that, then follow the instructions and you will have windows xp running on a SATA drive.
