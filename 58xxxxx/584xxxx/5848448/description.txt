Sierra's "Fast Attack: High Tech Submarine Warfare" Version 1.01

Included in this package:

1) FAST_ATTACK.iso - Disk image of the original CDROM. This game will install and run well using DOSBox.

2) fast_time.zip - Mod that extends mission time limits up to 6 hours rather than the default 2 hours. Follow the readme and change make sure to change the name of Ocean.dat to oceandat.

3) fa_train.zip - Community tutorial that details various important aspects of gameplay.

Be sure to visit the forums at http://www.subsim.com/radioroom/ for questions and discussion regarding this game and others.

You may read a review of this game at http://www.subsim.com/ssr/fastattack.html.
