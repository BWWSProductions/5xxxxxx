
Linux Mint 9 "Isadora" - Fluxbox CD (32-bit)

This is the Fluxbox edition of Linux Mint 9, codename Isadora, based and compatible with Lubuntu 10.04 Lucid Lynx and its repositories.

New features

Based on Lubuntu 10.04 Lucid Lynx, Linux 2.6.32, Fluxbox 1.1.2 and Xorg 7.6, this edition features a lot of improvements and the latest software from the Open Source World.

Featured improvements in this release: 30,000 applications catalogued and reviewable both online and in the new software manager, brand new incremental backup tool for both data and software selection, USB installer, 3 years support, look & feel improvements.
