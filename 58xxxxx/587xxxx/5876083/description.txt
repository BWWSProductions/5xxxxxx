
iPhone TomTom Europe v.1.4.1 Radar Safety Speed Cameras Map v.8.55.2885 ipa

This is a custom made ipa file of TomTom Mobile Navigator that covers Europe, with its latest version of 1.4.1 for iPhone, iTouch, and iPad, and with latest map version 8.55.2885 that included such features as: 

     1. IQ route
     2. Text to speech
     3. Lane assistance
     4. Google POI search
     5. Radar safety speed cameras with red light cameras at intersections

2. Radar Safety Speed Cameras with Red Light Cameras at Intersections
