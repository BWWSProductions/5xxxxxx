How do you browse a Blu-ray disc directly on your PC? You need to install the UDF file system. But where are you gonna find it? They don't make it easy, do they? And this is free software. So you don't have to go running around looking for it, here it is. This is Toshiba's UDF File System driver for Windows. Just right-click  thdudf.inf and choose "Install." Then reboot.
