Ubuntu Complete Remix 10.10 LiveDVD aims to have a much wider selection of software included, 2GB's worth! this release contains updated packages, and is installable. Notable inclusions: Adobe Flash, Java, Mp3 support, Firefox 3, OpenOffice Suite, Gimp, Pidgin (and Empathy), programmer IDE's, GParted and other rescue tools, text editors with syntax highlighting, and a collection of games.

This is the x86_64 version, visit http://www.ubuntucomplete.co.uk for more information.
