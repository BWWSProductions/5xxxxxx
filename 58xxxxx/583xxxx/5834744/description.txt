This is the CD from the book "Edexcel AS and A Level Modular Mathematics Further Pure Mathematics 1"
It includes an electronic version of the book and a solution bank. Also on the disk is an Exam Cafe which can help when preparing for an exam, it includes timetables, check lists and more.
File is in .iso form, you will need to mount the image on something like PowerISO, or use 7zip or similar to get all files out of the image.
