                            RockPlayer Universal Version. 1.1.42

                                                             
RockPlayer is an embedded platform, high-performance all-format video player. With optimized for mobile devices, and highly optimized core players FFMpeg decoder, RockPlayer Android platform has become the highest performance of the most 
extensive format support for video player.

What's New in This Release:

* Universal means run on all Android phone known. No more headache of CPU identify, the app handles it automatically.
* File type association : you may invoke RockPlayer from file manager app now.
* DropBox integrated.
* Subtitle charset choosing.
* Built-in update checking.
* A lot of small changes here and there.
