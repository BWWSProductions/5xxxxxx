Valkaama is a collaborative Open Source Movie http://www.valkaama.com/

These are all available media sources of the Movie in order to give you the chance to build upon them. If you want to participate in this project, by creating remixes, composing a new soundtrack, making a new trailer or if you want to publish anything else which is related to Valkaama, just get in touch. 

Within this package there are 632 video and audio source files. The formats used are M2T and WAVE. 

The filename convention for the videos is: xx-yy.m2t. Where xx denotes the cassette number and yy the take. M2T (MPEG-2 Transport Stream) is the format created by HDVSplit when transferring data from a HDV camera to the harddrive.
