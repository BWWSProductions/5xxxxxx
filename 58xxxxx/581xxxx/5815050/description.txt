                          Extended Controls Version. 4.0.1


                                                          

Extended Controls is a Control bar Widget with different colors and dimensions.

Version. 4.0 Changelog:

* Added new method for enable/disable Mobile Data network (APN)
* Added the Edit widget feature
* Added possibility to choose the switch position
* Added AudioManager toggle
* Added "Disable unused widget" feature
* Added AlwaysOn on Torch feature
* Added reboot on-demand
* Added brightness percentage label
* Added Russian locale
* Added timeout label (in second)
* Added label colors
* Added label color picker
