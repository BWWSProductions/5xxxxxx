                             SPB Mobile Shell v3.5.5 (Build 10087)



                                                            

Spb Mobile Shell introduces the next generation user interface while keeping all advantages of a Windows Mobile device.


Cool widgets to customize your Home screen:

* Photo contacts
* Picture frame
* Media player
* Weather
* Internet search
* Facebook and Twitter status
* Wireless switches
* Phone profile
* Appointments and tasks
*  ... and many others

Finger friendly contacts with Facebook integration:

* Widget-based photo contacts
* Facebook integration
* Photo call log
* Smart contact search
* 3D favorite contacts carousel
* Ringtone management

Fast access to all your smartphone features:

* Pinned shortcuts
* Recently used programs
* Finger friendly task manager
* 3x3 phone style menu

Spb Mobile Shell comes with integrated tools:

* Weather forecasts
* Agenda view
* World time
* Wireless Manager
* Profiles

Customize your device with your preferences:

* Make your own Home screen using widgets
* Adaptive skins
* Customize your menu
* Choose Professional or Lifestyle Home screen

What's New in This Release: 

* 320x480 resolution supported
* Fix: sometimes filtering and sorting in the Task widget didn't work
* Fix: it wasn't possible to restore settings from a file larger than 96K
* Other miscellaneous bug fixes and optimizations
