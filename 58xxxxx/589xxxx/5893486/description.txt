Open Torrent Tracker Project allows you to run your own open bittorrent tracker without the need of a html website, where conventionally people would upload torrent profiles that would have reflection to the hash database.

Version 0.03 is made a lot easier to get going and fixed a bug with a table name.

With OTTP you can say do your own http://publicbt.com/

Requires;
* server
* domain name (optional)
* php5
* mysql server
