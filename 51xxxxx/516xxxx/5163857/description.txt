This is noweb literate programming tool,
built for win32 (winxp) using mingw and msys.

Many standart filter build with Unicon 11.6 for Windows,
with some wrappers to launch it under msys shell.

Again, IT SHOULD BE RUN UNDER MSYS SHELL.
