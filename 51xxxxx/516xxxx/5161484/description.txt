You can convert multiple files in a single batch, regardless of their source and target formats. In addition, the program includes a built-in player. The audio format conversions are performed without any temporary files, allowing high conversion speeds. Other features include normalization (adjust volume), support for ID3 tags and more.

Note

If you want to convert RealMedia files(RM, RA, RMJ, RAM, RMVB) to other audio format files, you need install RealPlayer(RealOne) or Real Alternative on your machine. 
The conversion speed of RAM format file is slow, and it depends on the speed that you connects to the server linked by the RAM file. 
Please make sure that your RealMedia files could be played by your RealPlayer first. 
DRM protected WMA and M4B files cannot be supported, and please try the converter before you order it. 
Some .voc files are not Creative VOC format files, maybe they are RCA Voice files, and they cannot be converted to other audio format files. 
WAV format input support: PCM, MS-ADPCM, IMA-ADPCM, A-Law, u_Law, GSM610, Float-Double etc.
