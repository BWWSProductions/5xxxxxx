A package of payware items by Dutch creator AmazingMike. This package includes:

- NS 2400 pack complete
- Kirow KRC 1200
- NS Hippel package complete
- Strukton 1206
- V100 ITL pack 1 and 2
- V100 Shunter
- Talbot DR
- Talbot DB
- DB V100
- NS 345 Sik
- NS 6400 package
- NS Sik graffiti

All these items are build for Trainz Railroad Simulator 2004 and above.

Unzip or unrar the included items and install the .cdp files into CMP.

Have fun!
