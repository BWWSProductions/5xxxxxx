PHLAK is a Linux distribution Live CD based on Morphix, that focuses on providing network security tools. The acronym stands for Professional Hackers Linux Assault Kit. It includes security tools such as nmap, nessus, snort[citation needed], the coroner's toolkit, ethereal (now called Wireshark), hping2, proxychains, lczroex, ettercap, kismet, hunt, and brutus.

Installation Instructions:
Use PowerISO or any other ISO burning software to put image to disk. PHLAK can run solely from the CD or can be installed onto your hard drive. (Installs just like any other Linux based Operating System)
