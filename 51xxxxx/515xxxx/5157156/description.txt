My Wifi Tuner is a powerful application which helps you find the place with the best and the most stable signal and allows you to set the network configuration according to security standards. 

 
With a user-friendly Start Assistant you can set up a network connection in several clicks. The Security Assistant provides useful tips to help you create a safe network and protect your critical information. 

The program consists of the following fields:  
1)Toolbar with easy to understand icons 
 
2)Network Field shows all available adaptors and access points 
 
3)Status Field displays complete status information as well as graphic displays of network activity 
 
4)Explorer Bar allows you quick access to the main program features
