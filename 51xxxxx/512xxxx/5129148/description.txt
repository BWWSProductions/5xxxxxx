Installation notes:


  1. Mount the .iso with DAEMON Tools Lite v4.30.4 : 
     http://www.daemon-tools.cc/eng/downloads

  2. You might need to set daemon tools in the Preferences -> Advanced -> select all: RPMS,Securom ...
  
  3. Install the Oxford Dictionary with the Full Instalation option/setting.
     Do NOT run the program  immediately after installation !

  4. Copy the patch_OALD7.exe file from the crack folder to the installation directory/folder (where the dictionary is)

  5. Run/install patch_OALD7.exe

  6. Copy and overwrite oald7.exe from the crack folder to the installation folder

  7. Run oald7.exe -> finished.
