WinMount is a useful windows utility. It is an archive tool and also an image tool. It can compress, decompress and browse archieves, mount archives and DVD CD HDD images to a virtual disk. The most powerful function is that mount ZIP RAR to virtual disk or local folder directly without extraction. Supported formats: MOU ZIP RAR WIM VHD VDI VMDK ISO ISZ BIN MDS/MDF NRG IMG CCD CUE APE FLAC WV WAVE.
