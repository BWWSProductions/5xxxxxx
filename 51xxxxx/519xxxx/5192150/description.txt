English

JkDefragGUI is a Graphical User Interface which allows you to easily select all the features of JkDefrag, a freeware 

defragmentation tool made by Jeroen Kessels has to offer. But it can do a lot more. JkDefragGUI can schedule a 

defragmentation, install JkDefrag or its screensaver but can also do some small maintenance to your computer. It can be 

used to schedule a defragmentation of your system files like pagefile and registry files. JkDefragGUI also supports 

CCleaner which can be used instead of the internal cleaning function. It can also erase any information which is left behind 

on your hard disk after erasing files. JkDefragGUI was created for people who don't have the skills of an advanced user 

but still want to use the powerful features JkDefrag has to offer or for those people who like a GUI instead of using the 

command line. JkDefragGUI is compatible with Windows 2000, Windows XP, Windows Vista, Windows 2008, U3 capable 

flash memory and with BartPE pre-installed environment.
