├é┬▓  Release Description:
├é┬▒  ~~~~~~~~~~~~~~~~~~~

   RAR Password Recovery Magic is a powerful tool designed to recover lost
   or forgotten passwords for a RAR/WinRAR archives. 
            
   RAR Password Recovery Magic supports the customizable brute-force and 
   dictionary-based attacks. 
          
   RAR Password Recovery Magic has an easy to use interface. All you need
   to do to recover your password is just to add your file to the operation
   window.
