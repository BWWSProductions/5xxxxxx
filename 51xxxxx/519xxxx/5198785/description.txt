                           Description                            
                                                       
                                                                        
                                                                          
      alarmToday - Your complete alarm control center, never overlook     
                        or overhear an alarm again.                       
         Supports Windows Mobile WM 5.0, WM 6.0, WM 6.1 and WM 6.5        
                                                                          
      alarmToday is a Today screen plugin - giving you total control      
      over all alarms on your Pocket PC. In addition you can now also     
                run alarmToday as a stand-alone application               
      You can view, change, delete and create new alarms with just a      
                                few clicks.                               
       Always have repeating alarms, wake up to your favorite music       
     every morning or turn-off all sound while in a meeting. These are    
             just some of the many possibilities of alarmToday            
     Together with buttonMax you can execute macros to perform complex    
      tasks unattended. For example automatically connect to Internet     
          and download Email / AvantGo content during the night.          
                                                                          
                      30 day trial version available.                     
                                                                          
                          alarmToday Key Features                         
                                                                          
             Today plugin with overview of ALL pending alarms             
         Create quick alarm notes (hand drawn, text or recording)         
         Powerful custom alarms, play music, open a document or an        
                                application                               
                 Provides repeating and escalating alarms                 
               Override system sound so no alarms go unheard              
       Disable sound with a single click when you're in a meeting or      
                                  cinema                                  
     Set custom sounds per alarm type, also per category/priority for     
     Appointments/Tasks - up to 20 different sounds can be configured     
            Single click open / edit alarm in host application            
                       Easily delete unwanted alarms                      
                   Automatically remove duplicate alarms                  
      Fully integrated with Outlook, Pocket Informant & Agenda Fusion     
              Cleanup Temp directory or Web cache on startup              
                German & Spanish language support included                
                          Recurring custom alarms                         
               Execute macros on alarm (requires buttonMax)               
                   Set custom Task time (not always 8am)                  
         Create a mute alarm - mutes device at specified time for         
                             specified period                             
                                                                          
                                                                          
               New in v3.00:[Build 160] (26th November 2009)              
                                                                          
             WM 6.5 support added and WM 6.1 support improved             
         Host option to run alarmToday without Today Screen plugin        
              TGA and PNG alphablended image support in skins             
       VGA skins automatically scaled for QVGA screen or vice-versa       
     Settings in seperate application to reduce "In Memory" footprint     
                 Nav_Order support for complete 1-hand use                
                                Improved UI                               
                 Touch-Flo scroll bug in alarmNotes fixed                 
                  Improved screen refresh / rotate speed                  
          Improved cleanup of duplicate calendar / reminder items         
                 Fonts improved (more styles, and layouts)                
                 AppToDate support for automatic upgrades                 
      No longer installed in "My Documents" folder to resolve issues      
                              with ActiveSync
