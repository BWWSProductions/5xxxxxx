Unreal is a first-person shooter from 1998, developed by Epic Games, GT Interactive, and Digital Extremes.

It uses the Unreal Engine 1.

System Requirements:

Minimum CPU Type: Pentium
Minimum CPU Speed: 166 MHz
Minimum RAM Required: 16 MB
Minimum Hard Disk Space: 100 MB
Graphics Type: SVGA
Graphics Resolution: Multiple Resolutions
Color Depth: High Color

This game is rated: M for Mature - Animated Blood & Gore, Animated Violence

To install:
Mount and install with DAEMON Tools, PowerISO, or Alcohol 120%.

Online Play included.
