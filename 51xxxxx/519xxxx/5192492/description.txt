- Install game (when it ask for SID file you have to select it on your dvd drive).

- Enter on RAZOR1911 folder located on your dvd drive.

- Copy the content from UPDATE dir to your install folder and execute the rzr-init.exe to update the game.

- After update is complete, copy all files from CRACK folder to your main installation directory and overwrite.

- Enter on REG folder and select one of .reg files. Copy & Paste the .reg file into the installation directory (choose only 1 depends on your OS), if it's 32-bit, then choose the 'x32'. if it's 64-bit, then choose the 'x64'.Once copied, double click the .reg file, choose 'YES' when prompted.

- PLAY






If you want to play on multiplayer, play on Garena
