LucasArts Games Collection
==========================
Here is collection of some great LucasArt's games:

* Day Of The Tentacle
* Full Throttle
* The Secret Of Monkey Island (Monkey Island 1)
* Monkey Island 2: LeChuck's Revenge (Monkey Island 2)
* The Curse Of Monkey Island (Monkey Island 3)
* Sam & Max Hit The Road

All of them are built on top of SCUMM game engine, that is why you have
to use ScummVM program. This collection provide all necessary files
to let you play this games using ScummVM.

I included ScummVM for Win32 platform and it's source code for using on
other platforms (as majority of users still use Win32).

All files compressed using powerfull free software compressor -- XZ. I
also included it's binaries and source code here.
