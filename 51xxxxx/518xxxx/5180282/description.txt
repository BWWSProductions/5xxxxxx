This is the Hanthana Linux standard edition.
Hanthana Linux is a Fedora respin Live CD that can be run straight off the CD or installed to the local PC.

Hanthana Linux is aimed at low end PCs as well as high end PCs. Also its loaded with all software that are targeted to school and home users alike. One of its primary goals is to have all necessary software on the CD itself so that it can be deployed on PCs without internet access and still have a fully working desktop.

For more details about the project please see : www.hanthana.org

This version has the software updated to latest stable versions.
