This is the Hanthana Linux SOHO server edition.
Ideal for Small Office/Home Office environments.

It provides :

Apache2 Web Server
Postfix SMTP server
Dovecot POP3/IMAP server
Roundcube Webmail
Squid proxy server
Dansguardian web filter
Clamav virus scanner
Spamassassin spam filter
Samba server
MySQL database server
phpMyAdmin
CUPS printer server with printer drivers
SSH server
ProFTPd FTP server
PPPoE support out of the box

Hanthana Linux is a Fedora based respin. See www.hanthana.org for details
