Version 0.3

Tool to download DRM-Free content from BBC iPlayer. This content can be played on any device that supports Apple QuickTime Movies (.mp4 files) and MPEG-1 Audio Layer 3 (.mp3) including iPhone, iPod Touch and any PC with the latest QuickTime Player installed.

Supports TV and Radio programmes, although not all programmes are available. Search BBC iPlayer programmes based on Channels, Genre and Title search.

The latest versions have restrictions allowing only 1 download unless licenss is bought.
