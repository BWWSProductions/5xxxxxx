Core Player S60v3 + Keygen + PhoneInfo

A great media player to watch DivX and listen to FLAC on your Mobile (Nokia, Sony Ericsson and more)



1. Install Core Player and PhoneInfo(not necessary, see below)
2. Run PhoneInfo and write down your IMEI number
   (or look behind the battery, the IMEI number is usually printed there) 
3. Run keygen on your pc and generate code
4. Run Core Player and enter code
