Product Description
SwitchTop is an application that allows you to have 
multiple desktops. Is your desktop cluttered with many 
files and documents? Do you wish you had more desktop 
space? SwitchTop is your answer. Easily create and 
manage your desktops and switch between them quickly. 
SwitchTop is simple to use, all of its features are 
accessible through it's application dock menu.

What's New?
--> Change the desktop picture straight from the dock

Installation:
--> Open folder & unzip file
--> Drag app to your applications folder
--> Use keygen (.exe) to generate serial
--> Register program with serial
--> Done!
Note- Keygen is a windows app. You must have a 
copy of windows to run the keygen. SwitchTop is a mac 
app. When I have time, I will create a mac version of the 
keygen...
