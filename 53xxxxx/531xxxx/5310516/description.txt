Seven Remix XP adds in Windows XP a new style of Windows 7. Now switch to the new style in the parameters of "appearance of windows, you get a stylish taskbar of Windows 7. Also added new wallpapers and a welcome screen with the logo of Windows 7 Ultimate at the entrance to the OS. You can return to the old style XP simply switching to the parameters of "R" on Windows XP. Program can be reinstalled after OS updates.
