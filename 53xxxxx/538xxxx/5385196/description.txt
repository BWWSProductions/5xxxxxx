This is the moonshell_10.DPG Converter this converts the supported files into DPG to be watched via moonshell on your R4 card or another system that supports it.

The supported files are .wmv, .mpeg, .mpg and .avi

To open the converter open 'dpgenc' and click 'run'. Then to select the file click 'select' and choose the desired file and just click 'start' to start the conversion.

The conversion will take longer to start depending on the size of the file and it's duration.
