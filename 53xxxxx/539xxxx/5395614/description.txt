Need for Speed - Hot Pursuit 2 for PC.
Rated E for Everyone.
After downloading, look for the readme file. It contains info on how to use this file. The readme file contains the serial code.

Default Keyboard Driving Controls:

Up Arrow = Gas
Left / Right Arrow = Steer
Down Arrow = Brake

Spacebar = Hand Brake

C = Camera View
R = Reset Car
B = Look Behind
A = Shift Up
Z = Shift Down
H = Horn (Siren/Target Speeder in Be the Cop Mode)

U = Call for Backup (Be the Cop Mode)
J = Call for Roadblock (Be the Cop Mode)
N = Call for Helicopter (Be the Cop Mode)

ESC = Pause
