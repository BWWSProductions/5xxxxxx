CorelDRAW Graphics Suite - a well-known software package for working with different types of graphs. CorelDRAW Graphics Suite allows you to work on all sorts of projects - from logo creation and Web graphics to multipage marketing brochures and eye-catching signs. The package CorelDRAW Graphics Suite implemented advanced means of ensuring compatibility with commonly used formats, including Adobe Illustrator, PhotoShop, Corel Paint Shop Pro, Microsoft Office, JPEG and PDF. This publication provides a closed beta version of the software bundle.
