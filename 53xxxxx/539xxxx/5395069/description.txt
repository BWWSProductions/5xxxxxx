This is a virtual machine created by VMWare 5, installed with Debian 5, mono, MySQL and Opensimulator. This allows you to run an opensim server, and connect to it with a viewer on your OS with a minimum of effort. What is opensim? Check out http://opensimulator.org/ for details. Also see what you can make here:as an example. Please read the README before attempting to run this virtual machine, for user login info. You'll need at least 4GB on your hard drive to unzip this archive.

Enjoy!
