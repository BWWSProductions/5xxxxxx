// ** What is Anthies_Smiley_Pack ** \

- It's a smiley package for phpBB3 websites. 
- Can be used for MSN too.



// ** What is it for? ** \

- To refresh your phpBB3 board.
- To get the most popular and best smileys ever.



// ** How do I install them? ** \

- You just need to open the install.txt file in the anthies_smiley_pack folder.
- Follow the instructions and you will be successful.



// ** How do I contact the author of this package ** \
