A few days ago I downloaded a siterip of this interesting high-definition Asian massage-themed site. The videos were ostensibly 1920x1080 Full HD, but when the downloads were all finished I noticed the aspect ratio was all bunged up -- 1440 x 1080, which is not the 16:9 of Full HD. Everything looked stretched out. Bummer.

But! I've got a brand-new Core i7 iMac. With all four processors blazing, Handbrake re-encoded all the WMV clips to high-quality x264 in just a little over five hours. Unfortunately I had to take the resolution to 1440 x 810 (16:9 aspect ratio) because Handbrake won't up-scale. So they're no longer Full HD (and probably not at the same quality as the original clips on the site, had the aspect ratio not gotten fucked up), but each file is now only about 40% the size of the original files.
