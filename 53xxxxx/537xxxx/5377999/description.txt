This is a text file with the script for CEX Clone 3.0 for messenger plus. You NEED Messenger Plus installed first before you can use this script! 

To use it go to show menu on the messenger window > 

scroll to plus > 

click preferences and options > 

on the general tab click scripts on the left hand side > 

then in the developers only box click create new >

enter the name CEX Clone 3.0 then delete everything in the window that appears > 

then open your text file and copy and paste the entire text into the script window> 

then click save all at the bottom and your done :-D

Enjoy
