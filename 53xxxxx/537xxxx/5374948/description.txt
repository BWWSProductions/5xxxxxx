CONTROLS:
Aim: Mouse Movement
Fire: Left Mouse Button
Sniper: Right Mouse Button
Previous Weapon: Q
Game Paused: P
Hint: H
You can customize the default controls by selecting "Setup" at the Main Menu.

Weapons: Use the number key listed below to select a specific weapon.

M1911: 1
M16: 2
Mortar: 3
Mini Gun: 4
R.P.G. Rocket: 5
Artillery Strike: 6
Napalm Strike: 7
Huey: R
Enjoy!
