This is the Swedish edition of the game Championship Manager 97/98.

Believe it or not but it's a ~300 MB ISO compressed to a ~5MB .RAR

This works best on Windows 95/98/XP 32bit. Use DOSbox in 64bit, Vista and W7.

1. Unpack
2. Mount
3. Install (to C: and then move it to wherever you want)
4. Play

Windows 7 32-bit:
This works on W7 32-bit if you either use the default video driver or the driver made for XP(this option might cause your machine to become unstable).
