This installer will install the following games,

Carnivores (1998)
Carnivores 2 (1999)
Carnivores Ice Age (2001)
Carnivores Cityscape (2002)
Primal Prey (2001)
Shark! Hunting the Great White (2001)

Primal Prey and Carnivores Cityscape have no cd cracks already applied and Carnivores, Carnivores 2, and Carnivores Ice Age have been patched up. Also added the installer for a glide wrapper i found online, will help if you want to run the carnivores games in 3dfx glide.
