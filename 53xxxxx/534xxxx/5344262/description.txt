PwnageTool 3.1.5 is a Mac based iPhone firmware creation tool for creating custom ipsw/firmware for various Apple handheld devices. 
This version supports the 3.1.3 firmware from Apple for a specific set of devices that are in a specific technical state. 

PLEASE read the blog post at http://blog.iphone-dev.org/post/376648600/pre-game-show before attempting to use this software, real mistakes can be made that could disable features (such as a carrier unlock or ability to jailbreak) in your device.

For full details of any of our released applications and techniques, please visit the iphone DevTeam blog. http://blog.iphone-dev.org
