RMclock (Rightmark CPU Utility) 2.35 with keygen


Description
---------------------
This is also called RMclock. It can be used to control your processors speed settings via a system tray icon. 

Works with windows vista and windows 7.

Installation
---------------------
You should move this to its own folder (c:rmclock) or something similar. 

Right click rmclock.exe and create shortcut, then move shortcut to your start menu in the 'startup' folder
