Call of Duty 4 Modern Warfare 2 Keygen

Last checked on January 18, 2010


1. Download the file

2. Run the Database Check Tool

3. Run the Keygen

4. Copy the key

5. Open regedit (Start - run - regedit)

6. Navigate to HKEY_LOCAL_MACHINE-Software-Activision-C all of Duty 4 Modern Warfare 2

7. Click codkey and change the value to your new serial key

8. Enjoy :D
