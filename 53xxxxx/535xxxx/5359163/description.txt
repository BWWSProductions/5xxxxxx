Ford DVD Navigation System 2008 - 2009

The file 464410-5280.iso contains the firmware for the Navigation system.
This file must be burned on a regular CDROM. When this CD is inserted in the 
navigation system, download of firmware will -if required- will start
automatically.

The file 464210-9961.iso contains the 2008 - 2009 maps of Western Europe.
This file must be burned on a dual layer DVD (DVD+R DL).

The file's source is from originals as obtained from the Ford dealer in
February 2010.
