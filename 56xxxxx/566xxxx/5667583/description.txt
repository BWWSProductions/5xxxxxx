Garmin BaseCamp Special version 3.05

This modified executable will let you use old type (only 25 digit key) unlocking mechanism.

This version doesn't need .gma unlock files but accepts unlock keys also for
FIDs above 2200, e.g.: CN Europe 2011.10 and CN North America 2011.10.

Instructions for use:
- Install BaseCamp 3.05
  (get it from Garmin website: (see link in .nfo) )
- Replace BaseCamp.exe
