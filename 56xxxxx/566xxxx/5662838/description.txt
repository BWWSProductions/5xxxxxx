Instructions:
=============

Mount the included DMG file with Roxio Toast or burn it to a dvd.

Drag the 'RAP' folder to your HD.

Run Adventure in Green Valley.app

Enter the provided serial (serial.txt) when asked.

Requirements:
=============

Macintosh (G3 or higher recommended)
MAC OS 8.6 to 10.x
32 MB or more of installed RAM

If classic environment is no longer supported, this game will not work.
