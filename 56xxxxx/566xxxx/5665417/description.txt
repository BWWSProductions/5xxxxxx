Little .exe mod that makes Settlers II 10th A. a portable version. Enjoy.




REQUIREMENTS :::::::::::::::::::::::::::::::::::::::::::::::::::::


(Skip it if you already have "Settlers II 10th Anniversary" installed.)

1. Get "Settlers II 10th Anniversary" from any source (shop, p2p).
	
2. Install "Settlers II 10th Anniversary" on your computer according to instructions provided by source.
2.1. You can install game on flash drive. In that case, remember to skip step 1. in PORTABLE INSTALLATION
	
3. Make sure that your "Settlers II 10th Anniversary" works fine (I don't want to solve problems given by your "Settlers II 10th Anniversary" version or bad installation)
	
PORTABLE INSTALLATION ::::::::::::::::::::::::::::::::::::::::::::

1. Copy just installed game directory (eg. X:/Program Files (x86)/Ubisoft/Funatics/The Settlers II - Anniversary) on your flash drive.
	
2. Copy "S2DNG_Launcher.exe" file to that directory (on flash drive of course).
2.1. File "S2DNG_Launcher.exe" MUST to be placed in root OR 'bin' directory.
	
3. From now on run game ONLY by "S2DNG_Launcher.exe" file! In other case, your saved games will not update on flash drive.
3.1. Play "Settlers II 10th Anniversary" everywhere you like.
		
FAQ ::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::

What happen with my old saved games from "Settlers II 10th Anniversary" if I install portable?
		
If you do everything right while PORTABLE INSTALLATION, your saves will be copied on flash drive on very first start of "S2DNG_Launcher.exe".
		
If I playing on portable AND normal installation can I lose my saved games from "Settlers II 10th Anniversary"?
	
Yes. Once you install this portable it is strong recomended that you play only portable version.
Playing alternately both versions may result that saved games maked by normal installation will be overwrited (portable saves has higher priority).
		
What meaning of life has "S2DNG_Launcher.exe"?
	
It is responsible for keeping saves on flash drive (or any other place where that portable version will prosper).

MOD WORKS ONLY UNDER POLISH AND ENGLISH WINDOWS (XP, Vista, 7; x86, x64)
