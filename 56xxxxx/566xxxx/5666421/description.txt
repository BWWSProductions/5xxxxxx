Router/Bridge Linux Firewall
Zeroshell is a Linux distribution for servers and embedded devices aimed at providing the main network services a LAN requires. It is available in the form of Live CD or Compact Flash image and you can configure and administer it using your web browser.
It's available the new release 1.0.beta13 of Zeroshell that includes the following new features and improvements: an accounting module preview for controlling the time and the bandwidth per user connection; the Bandwithd application for checking per host bandwidth; some bug fixes.
