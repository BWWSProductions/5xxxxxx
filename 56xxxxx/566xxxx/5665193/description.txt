It's a speedway simulator. People from some countries play this game, start in the league and compete for champion of Polish League STS. Over 1000 users register on polish forum. Download and play !

System Requirements:
OS: Windows 98/2k/XP/Vista/7
CPU: Pentium II 400MHz
RAM: 64MB 

Password: stspl.eu


Installation:
1. Install TurboSliders-2.0.0.exe
2. Install TurboSliders_Speedway4.exe into folder with Turbo Sliders
3. Start play..

How play:
1. Start Speedway Turbo Sliders 4 (shortcut)
2. Network Game
3. Server List
4. Connect with server

or select some tracks and click Train ;)
