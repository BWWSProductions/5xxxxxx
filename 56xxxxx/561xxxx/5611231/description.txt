SmartSokoban v0.99.H.5 by VVitaliy a.k.a. Smarty

The main feature that sets this game apart from other similar games is its intelligent control system (ICS). With a couple of mouse clicks you do things that would take up to a couple of HUNDRED key presses to do. Using the ICS is simple: Just click the box and then click a destination square. If the box can be moved there it will be done.

SmartSokoban supports skins. New levels can be easily added.
