essential tools for psp gaming

copy all files in your Ms root and restart you psp once.
you can switch between 5.00m33-6 and Gen D3
any time you want.

if you are on m33 use gen D3 install
  select Flash Install CFW 5.50 GEN-D3

if you are on gen use recovery flasher v1.60 tool
  just select "Install CFW (of version selected above)"
  make sure the above line is "Target FW: 5.00"
  that's all.
