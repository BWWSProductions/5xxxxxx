Originally developed by Ken Fern the database is about useful, medicinal, edible plants around the world.

Plants for a Future or PFAF for short.


It is an SQL file with README.txt and a Web database.



The SQL table is organized in the same fashion as found on pfaf.org


This torrent might be important to people interested in, forest gardening, permaculture, bioengineering, seedballing, nature, environment, botany, agriculture, survivalist, intentional communities.
