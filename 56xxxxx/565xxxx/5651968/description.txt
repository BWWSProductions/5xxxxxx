MagicISO Maker is a powerful tool for CD/DVD image file creating, editing and extracting.
It can backup CD/DVD to your hard disk and directly create/edit/extract ISO files. It can make bootable CD image. 

MagicISO Maker also can convert BIN or any other format file created by CD burning software
