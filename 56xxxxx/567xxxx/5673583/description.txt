:: Tweak Pack for PowerDVD 10 ::
================================

Blu-Ray_Region_Reset.bat:         Resets the Blu-Ray region settings
Disable_BD_Mouse_Pop-up.bat:      Disables the Blu-Ray disk Mouse support Pop-up
Enable/Disable_3DFeature.bat:     Enables/Disables the 3D Feature
Enable/Disable_MoovieLive.bat:    Enables/Disables MoovieLive, Remix + Comments
Enable/Disable_CinemaMode.bat:    Enables/Disables the Cinema Mode
Enable/Disable_CLFLVSplitter.bat: Enables/Disables the FLV Splitter *
Reset_HD_Dailog.bat:              Resets the High-Definition Content dialog **
Remove/Restore_StartupApps.bat:   Removes/Restores the Startup Applications ***
Enable/Disable_Toolbar.bat:       Enables/Disables the PowerDVD Toolbar
Enable/Disable_Upgrade.bat:       Enables/Disables the Upgrade Popups and Buttons


*    CLFLVSplitter.ax can cause a crash when you play flv videos in Windows Media
     Player. Disable_CLFLVSplitter.bat unregisters the FLV Splitter, which fixes
     the problem

**   This comes in handy if you selected "Don't ask me this question again" and 
     want to re-enable or disable TrueTheater Lightning and Smart De-interlace
     when playing High-Definition content

***  Enables/Disables BDRegion and RemoteControl10 from running at startup.
     Don't disable them if you're planning to use a remote to control PowerDVD!
