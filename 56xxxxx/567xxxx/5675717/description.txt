Skillsoft CompTIA A Plus Essentials 2009 Expert Encore

Overview/Description
The CompTIA A+ Essentials course provides students core knowledge required in the assembly, configuration and troubleshooting of PC computers. This course focuses on all official exam objectives of the CompTIA A+ Essentials examination, which has a focus on the standards and operation of PC components. In order to attain the A+ certification students must pass both the A+ Essentials 220-701 and A+ Practical 220-702 examinations. This course focuses on the A+ Essentials examination.

Target Audience
This course is appropriate for students pursuing the A+ examination. This course would also be helpful to anyone in the IT field that lacks PC computer troubleshooting skills. Students considering Microsoft administration should consider attending this course.

Prerequisites
There are no prerequisites for this course.

Outline
CompTIA A+ Essentials (2009 Edition)

    * Session 1: Motherboards, Cases, Power and Cooling
    * Session 2: CPUs, RAM and the BIOS
    * Session 3: Drive and IO
    * Session 4: Ports and Networking
    * Session 5: Windows Operating System
    * Session 6: Printers and PC Maintenance

Expected Duration (hours)
30.0

Lesson Objectives

CompTIA A+ Essentials (2009 Edition) Expert Encore

# Identify the function and operation of PC components
# Describe the standards of CPUs, motherboards, RAM and ports of a PC
# Explain the purpose of PC components
# Describe the basic configuration of PC components for operation
# Install, configure and maintain the Windows operating system on a PC
# Describe important network configuration for a PC
# Describe operation and configuration of printers
# Clean and maintain PC computers
