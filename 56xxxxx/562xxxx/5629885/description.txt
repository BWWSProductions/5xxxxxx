This the Norton Technician Toolkit, it is a portable scanner and virus removal tool, used by Staples Easytechs. Also has Tune up tools as well on it. 

It's commonly used to preform the Free Tune up that is offered at Staples. 

Works with Windows XP/Vista/7



How To Use

-Simply Extract the Zip to any Flash Drive (recommend at least 1GB)


-Open NTT folder and run Ntt.exe and you will see 3 options


-First is downloading the latest definitions make sure the option is checked and press execute


-Next select copy the definitions to flash drive and execute

-Now you can select configure scan and begin running your pc tune up.
