Dragonica-hack is an open source tool designed to help you with modifying Dragonica in Windows. You can make it harder or easier depending on your preference, but this also contains other usefull tools to help debugging Dragonica.

It comes with a memory scanner to quickly scan for variables used within a Dragonica and allow you to change them, but it also comes with a debugger, disassembler, assembler, speedhack, direct 3D manipulation tools, system inspection tools and more.
