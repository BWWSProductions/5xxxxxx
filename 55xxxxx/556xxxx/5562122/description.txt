Torrent Searcher - program-search engine on Torrent-loader networks and found files. It is possible to find music, movies, graphic files, programs and any other files in the networks of Ares, Fasttrack, OpenFT and Gnutella.

The program can create playlists from music and movies, supports preview downloads and viewing sketches (thumbnails) pictures. There is a search filter.
OS: Windows All
Language: English
