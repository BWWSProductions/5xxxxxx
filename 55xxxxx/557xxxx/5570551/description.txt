Starcraft Brood War 1.16.1 with no cd cracked exe
Auto extracting SFX zip archive, extracts to C:tempStarcraft
You can take the extracted copy from temp and keep it anywhere else to keep it as an installed copy if you want.
Maps included: Generic SC, Generic BW, iCCup maps+ obs vers and ~80 custom UMS maps I thought would be good.
I made it mainly for a portable LAN copy but if you have a working battle.net key you can open tempstarcraft and inside the tools folder you can move Battle.snp to starcraft and then use the key changer to modify the installed key to a working one.
It is installed with 3333 33333 3333.
The key changer is strange; to modify the key you choose the folder the starcraft folder is in, not the starcraft folder itself.
Just to note, this does not include the single player campaign.
