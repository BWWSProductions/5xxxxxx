To play this game:-

1. Extract this RAR file with any extractor (ex. WinRAR).
2. In this folder you find a "Registration.reg" file, Double click on that to Register your game in the PC.
3. Then there are a file Named "Play.bat", Double click on that file to Run the game.


4. When you run the game,
   When you see "namco" written in red color and start "Movie" then Press "V" button and then
   press "S" button to start.

	Keys are:

	S - To Small Punch (Sword)
	D - To Big Punch (Sword)
	X - To Small Kick
	C - To Big Kick
	Z - To Block
   
   That's all.
