IPhone SDK 3.2 provides support for application development iPad. iPhone SDK 3.2 includes a set of tools Xcode 3.2.2, compilers and freyvorki to create applications for iPhone OS and Mac OS X. Among other things, these tools include the Xcode IDE and analytical tools.
With this program you can create applications that run on iPad using the included iPhone Simulator with iPhone OS 3.2. In addition, you can develop applications for iPhone and iPod touch with iPhone OS 3.1.3. (This software does not include all the iPhone OS 2.x SDK). To install the iPhone SDK 3.2 requires a Macintosh computer running Mac OS X 10.6.2 (Snow Leopard) or later.
