ZOOMREADY .com * Project de Vinci Release
a project to release some print resolution 500 dpi sections of The Last Supper by Leonardo de Vinci.

These source PNG files can be printed life size or larger and maintain clarity. 
These files have been sized to match the approximate life size in the original 
painting in the Santa Maria delle Grazie, Milan Italy, roughly 500dpi.  
The window is 30" x 30", and the knife 15" x 15"
