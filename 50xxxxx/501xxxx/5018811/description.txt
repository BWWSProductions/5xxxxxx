Security utility to restrict access to key features of Windows

1st Security Agent is an excellent password-protected security utility to secure Windows-based computers.

1st Security Agent works under any Windows platform and offers an administrative support for controlling which users are allowed to access your computer and the level of access each user may have.

You can choose to restrict access to lots of Control Panel applet functions, including Display, Network, Passwords, Printers, System, Add/Remove Programs, etc.

1st Security Agent is a powerfull password-protected security application that gives you the possibility to protect your computer.

You can also assign separate system profile folders to each user, providing each with their own custom Desktop, Start Menu, Favorites, My Documents, etc.

Additionally, you can: disable Start Menu items, hide local and network drives, disable the DOS prompt, boot keys, real DOS mode, Registry editing, taskbar, task manager, and network access, hide desktop icons, and much more.

You can apply password protection to Windows and restrict users to running specific applications only. Security restrictions can be applied universally or just to specific users.

1st Security Agent also supports Internet Explorer security that enables you to customize many aspects of the Internet Explorer Web browser.

It lets you disable individual menu items, prevent others from editing your Favorites, disable individual tabs in the Internet Options dialog, as well as specific settings from each tab.

1st Security Agent allows you to import and export PC security settings, and offers a flexible and complete password protection. You'll find the program interface very easy to negotiate.
