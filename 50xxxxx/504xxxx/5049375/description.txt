   Release Information:

   The Amiga computer, launched by Commodore in 1985, combined superior
   multimedia technology with ease of use, inspiring millions of the most
   creative and enthusiastic computer users ever.

   Amiga Forever is the award-winning Amiga preservation, emulation and
   support package brought to you by Cloanto, Amiga developers since 1986.

   Different editions of Amiga Forever blend high-quality software and
   original content with the ultimate set of videos to chronicle and let
   you experience firsthand the history, culture, challenges and passion
   behind the Amiga.

   New features in Amiga Forever 2009:

   All included games and demoscene productions are now in RP2 format
   (uses less disk space and makes it very easy to add or remove files)

   RP2 Manager to scan for and import/export content

   Built-in RetroPlatform Library (to recognize and configure games, etc.)
   with live updates

   Seamless, one-click integration of AmiKit and AmigaSYS add-ons lets you
   experience a broader variety of Amiga perspectives*

   New system ROMs (e.g. the 0.7 Kickstart required to run the software used
   by Andy Warhol in the 1985 Launch of Amiga demo)*

   Improved documentation and help (new Tip of The Day balloon, new
   documentation in Windows Help format)

   Emphasis on usability: interface and settings are improved, but remain
   consistent with the previous 2008 version and with C64 Forever (also see
   Tips for Upgraders)

   Tested to comply with "Compatible with Windows 7" requirements, with full
   support for Windows Vista and Windows 7, both x86 and x64 (possibly still
   the first and only Windows software with custom-shaped glass-enabled skin)

   Additional Notes:

   We decided to release this cool Commodore Amiga emulation tool, plus all
   it's additional Games, Demoes and so on. Someone out there would perhaps
   say this should be released under the Application section, since it's not
   directly a game, but more a tool.

   In a way they're right and we have seen previous Commodore Amiga editions
   get released that way, but then again it is a tool that emulates the
   Commodore Amiga machine, but yet it features a ton of games with it.

   As proud Commodore Amiga owners, back when we were kids, then the nostalgic
   moments and good memories with the breadbox came in first line. Just to
   make us remember how much fun it was and to be honest, then some of the
   included games are way more fun, than a large deal of the games relased
   these days.

   That's why we choose to release this under the Games section
