FlipAlbum Vista Pro

FlipAlbum Vista Pro is a novel album creator and image editor that allow the user to view and edit images resident in a folder by automatically creating a virtual photo album.

The photo album created works like a real one,
with pages that can be flipped and with page thickness on each side corresponding to the number of pages in the album.

The album, or created FlipBook allows the user the possibility to browse through his images, bookmark them, 
and perform various commands such as copy, move and edit between different albums in a convenient manner.
