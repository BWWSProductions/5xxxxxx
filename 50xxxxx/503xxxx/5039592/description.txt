Description
123 AVI to GIF Converter is a conversion tool that help you to convert AVI clips to user-defined GIF animation, and convert animated GIF images to an AVI file. The latest release fixed many AVI decode bugs and improved converting performance. The interface also be redesigned to make your works more easily and more efficiently.

Features
* Easy to use - With the totally redesigned interface you can finish your works more easily and efficiently.
* Convert AVI clips to user-defined GIF animation.
* Convert animated GIF images to an AVI file.
* Crop and resize the source image to make appropriate size GIF images.
