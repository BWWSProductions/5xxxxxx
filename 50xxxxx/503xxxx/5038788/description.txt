Installation:

For normal 1.6 version:

1. Decompress the file
2. copy (PES2009_EDIT01.bin) from save.normal folder to the save folder in documents
3. copy (unnamed_1.bin and unnamed_70.bin) from rv0c_e.img to rv0c_e.img in kitserver
4. copy (Leages Logos 128_2.bin and Leages Logos 256_1.bin) from cv0a.img to cv0a.img in kitserver
5. copy (unnamed_9.bin) from cv02.img.normal to cv02.img in kitserver
6. copy (bayern_318.adx) from cs0d.img to cs0d.img in kitserver
7. copy all files from cv06.img to cv06.img in kitserver


For Championship users 1.6EX:

1. Decompress the file
2. copy (PES2009_EDIT01.bin) from save.championship folder to the save folder in documents
3. copy (unnamed_70.bin) from rv0c_e.img to rv0c_e.img in kitserver
4. copy (unnamed_9.bin) from cv02.img.championship to cv02.img in kitserver
5. copy (bayern_318.adx) from cs0d.img to cs0d.img in kitserver
6. copy all files from cv06.img to cv06.img in kitserver
