The Matrix Effect Screensaver 

1- INSTALL The Matrix Effect.exe
2- Open the control panel display, choose "YZMatrix v1.00" from the screen saver list.
3- You can adjust the scrolling speed.
4- You can adjust the matrix density.
5- You can switch font size.
6- You can edit the messages to anything you like. 
7- Enjoy this amazing screensaver.
