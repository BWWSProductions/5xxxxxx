Age of Empires  is a history-based real-time strategy computer game released in 1997. Developed by Ensemble Studios the game uses the Genie, a 2D sprite based game engine. The game allows the user to act as the leader of an ancient civilization by advancing it through four ages this is both the age of empires and rise of Rome Expansion
