The Great Pattern Harvest (vol 1) DEMO

This demo lets you try out the Great Pattern Harvest, which revolves around crochet patterns harvested from around the web.  The demo version lets you get a feel for how the search/viewing application works, and includes thumbnail images for all patterns.
