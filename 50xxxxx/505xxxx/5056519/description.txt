Open PowerISO, or MagicISO or whatever you use.
Click on "tools".
Then click "Compress CD/DVD image".
Select this WinRar archive when browsing for the file to compress.
At this point, make sure you select .uif for the output file. 
You will end up with a UIF file.
You can then mount it and run setup and play.
Enjoy.
