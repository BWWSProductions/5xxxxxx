Fedora-11-Multimedia-Live-Mega

It is a fedora-11 respin with full Multimedia support including realplayer, gnome-mplayer, xine,Vlc,Banshee, caffine, juk, fully operational totem media player amarock and a dozen multimedia applications.
it boots into gnome as well as kde desktop
it can be run from DVD in live mode
it can also be run from hard drive in live mode. it can be installed to hard drive with ease all possible third party repositories are preconfigured 
it can be updated after installation using yum
