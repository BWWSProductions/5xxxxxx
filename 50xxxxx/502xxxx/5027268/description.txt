The server install CD allows you to install Ubuntu permanently on a computer for use as a server. It will not install a graphical user interface.

If you wish to install a GUI, either use the normal desktop edition, or:

`sudo apt-get install xfce4
then
`startx



This is the official Ubuntu torrent. If you do not trust it, get the .torrent off MIT's website.
http://ubuntu.media.mit.edu/ubuntu-releases/9.04/ubuntu-9.04-server-i386.iso.torrent
